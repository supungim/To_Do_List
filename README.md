# To-Do List (Java, JavaFX, MySQL)

A **desktop task management application** built using Java and JavaFX, with MySQL for data storage. Allows users to create, update, and manage tasks efficiently with a responsive user interface.  

## Features

- **Task Management**: Create, edit, and delete tasks.
- **Due Dates**: Assign due dates to tasks and view upcoming deadlines.
- **Responsive UI**: User-friendly and intuitive interface using JavaFX.
- **Persistent Storage**: Tasks are stored in a MySQL database.
- **Search and Filter**: Easily find tasks based on title or due date.

## Tech Stack

- **Frontend**: JavaFX
- **Backend**: Java
- **Database**: MySQL
- **Build Tool**: Maven / Gradle (optional)
- **IDE**: IntelliJ IDEA / Eclipse (recommended)
